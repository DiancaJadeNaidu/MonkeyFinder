# Monkey Finder App (MAUI)

## Overview
Monkey Finder is a cross-platform mobile application built using .NET MAUI in Visual Studio. The app allows users to browse and view information about different monkey species, along with their locations around the world.

## Technologies Used
- .NET MAUI
- C#
- XAML
- Visual Studio

## Features
- View a list of monkeys
- Display detailed information
- Geolocation support to find the closest monkey to you
- Internet connectivity integration for fetching data
- Cross-platform supper (Android, iOS, Windows)
- Clean and responsive user interface

## Getting Started
### Prerequisites
- Visual Studio 2022 or later
- .NET MAUI workload installed

### Run the App
1. Clone or download the repository
2. Open the solution in Visual Studio
3. Select your target device (Android emulator, Windows, etc.)
4. Click run
